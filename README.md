## SETUPNYA MUDAH KOK

- **Siapkan Server anda menggunakan Laragon disarankan**

- **Buat Database terlebih dahulu**

- **Konfigurasi file ENV terlebih dahulu dan sesuai kan dengan database kalian**

- **Lalu jalankan php artisan migrate**

- **Lalu jalankan php artisan db:seed**

- **Setelah Itu Jalankan php artisan serve**

aplikasi kalian siap di gunakan (http://localhost:8000)
